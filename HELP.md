# Getting Started
Тестовое задание по созданию Rest сервиса с использованию Spring boot как сервиса агрегатора с использованием асинхронных вызовов.
Коннекторы к вненим системам не реализованы и только имитируют запросы.
Пример даёт общее представление о реализации подобного подхода.
